# 🖥️ Мои конфиги Arch + bspwm

Добро пожаловать в мой личный репозиторий конфигурации.  
Здесь лежат настройки окружения, которые я использую каждый день.

---

## 📦 Состав

| Компонент | Назначение |
| :--- | :--- |
| **bspwm** | Оконный менеджер |
| **sxhkd** | Горячие клавиши |
| **polybar** | Статус-панель |
| **rofi** | Запуск программ |
| **kitty** / **alacritty** | Терминалы |
| **neovim** | Текстовый редактор |

---
```
Сочетания клавишь
| **Терминал** | Win + Enter |
| **Меню приложений/rofi** | Win + D |
| **polybar** | Win + P |
| **Переключение между раб.столами** | Win + 1-6 |
| **dmenu** | Win + X |
| **firefox** | Win + Shift + B |
